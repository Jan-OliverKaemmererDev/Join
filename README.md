Join is a web-based Kanban project management tool developed as a group project at the Developer Akademie.

## Features
- User registration, login & guest access
- Dashboard with task overview and deadlines
- Kanban board with drag & drop
- Tasks with priorities, categories & subtasks
- Contact management
- Responsive design (desktop & mobile)
- Legal Notice & Privacy Policy

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Multi-Page Application (MPA)

## Project Notes
- Implemented according to the official Join project checklist
- Clean code and modular structure
- Manually tested in all major browsers
